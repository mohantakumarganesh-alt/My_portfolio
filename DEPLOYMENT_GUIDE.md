# Django Blog App - Render Deployment Guide

## Quick Start - 5 Minutes

### Prerequisites
- GitHub account (already connected)
- Render account (free tier available)

### Step-by-Step Deployment

#### 1. Sign up on Render
Visit: https://render.com
- Click "Sign up"
- Choose "Sign up with GitHub"
- Authorize the connection

#### 2. Create New Web Service
- Click "New +"
- Click "Web Service"
- Click "Connect repository"
- Search for "My_portfolio"
- Click "Connect"

#### 3. Configure Service
Fill in the following:

| Field | Value |
|-------|-------|
| **Name** | blog-app |
| **Environment** | Python 3 |
| **Region** | Oregon (or closest to you) |
| **Branch** | main |
| **Build Command** | `cd projects_code/blog_app && pip install -r requirements.txt && python manage.py collectstatic --noinput && python manage.py migrate` |
| **Start Command** | `cd projects_code/blog_app && gunicorn blog_project.wsgi:application` |
| **Plan** | Free |

#### 4. Add Environment Variables
Click "Add Environment Variable" for each:

```
SECRET_KEY = (auto-generated by Render)
DEBUG = False
ALLOWED_HOSTS = *.onrender.com
SECURE_SSL_REDIRECT = True
SESSION_COOKIE_SECURE = True
CSRF_COOKIE_SECURE = True
```

#### 5. Deploy
- Click "Create Web Service"
- Wait 5-10 minutes
- Once deployed, Render will show your URL like: `https://blog-app.onrender.com`

### Troubleshooting
- If build fails, check Render logs for errors
- Ensure all files are committed to GitHub
- Check that requirements.txt has all dependencies
