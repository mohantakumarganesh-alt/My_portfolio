Ganesh Kumar Mohanta - Portfolio Package
========================================

This folder contains the complete source code for my personal portfolio website and the included projects.

Directory Structure:
--------------------
/index.html            - The main portfolio website (Open this in your browser!)
/style.css             - Stylesheet for the portfolio
/script.js             - JavaScript for the portfolio
/demo-student.html     - Live web simulation of the Student Management System
/demo-blog.html        - Live web simulation of the Blog Application
/project-student.html  - Detail page for the Student Management System
/project-blog.html     - Detail page for the Blog Application

/projects_code/        - Contains the actual source code for the projects
    /student_management/   - Python/Tkinter source code (main.py)
    /blog_app/             - Django source code

Instructions:
-------------
1. Open `index.html` to view the portfolio.
2. Navigate to the "Projects" section.
3. Click "Live Demo" to see the web simulations.
4. Click "View Code" to see project details.
5. The full source code for the projects can be found in the `projects_code` folder.

Thank you!
